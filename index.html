03 / Working method

The publication follows the development of the work.

01Question

Start from a real uncertainty, failure, decision, or unexplained observation.

02Context

Inspect existing research, systems, source code, data, and prior attempts.

03Model

Define the objects, mechanisms, boundaries, and claims that matter.

04Build or analyze

Implement the system, run the study, or construct the comparison.

05Test and critique

Use evidence, failure, review, and counterexamples to revise the model.

06Publish

Record the result as code, a release, a note, a report, or a changed question.
